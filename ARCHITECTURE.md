# language-engine-witness architecture
This repository owns one capability: launch the four delivered runtime libraries as four separate OS processes, carry the real `spirit-min.schema` fixture through typed Unix-socket contracts, and independently compile the emitted Rust with the same rkyv dependency closure.

## Revisable lean
The independent crate supplies three legacy out-of-subset support types (`Topics`, `Kind`, `Magnitude`) while preserving byte-exact generated declarations for the six currently lowered items. Remove this support prelude when CoreNomos lowers the full fixture.
